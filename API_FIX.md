# 🚨 Quick Fix: API Key Not Working

## ✅ **SOLUTION: Use Demo Mode** (Recommended)

Demo mode works **100% offline** with pre-cached responses. Perfect for testing!

### How to Use Demo Mode:

1. **In the app, look at the top-right corner**
2. **Toggle the "Demo" switch to ON**
3. **Tap any quick example** (like "Make it more professional")
4. **Everything works instantly** - no API needed!

---

## 🔧 Alternative: Fix API Key (If you want live mode)

The API key needs to be validated. Here's how to check:

### Step 1: Test Your API Key
Visit: https://aistudio.google.com/apikey

Make sure your key:
- ✅ Is active and not expired
- ✅ Has "Generative Language API" enabled
- ✅ Doesn't have usage limits exceeded

### Step 2: If API is Valid
The issue might be that Gemini returns non-JSON responses sometimes. This is a known limitation.

**For hackathons and demos, ALWAYS use Demo Mode** - it's faster, more reliable, and works offline!
